.. _bluetooth:

Bluetooth
#########

This chapter contains information on the Bluetooth subsystem of the WM IoT SDK, which you can use to understand the Bluetooth protocol stack, the operation of the Bluetooth low-level controller, and its implementation methods.

The WM IoT SDK Bluetooth subsystem encompasses complete Bluetooth Low Energy (BLE) applications, from applications to RF transmission and reception, as well as partial support for traditional Bluetooth applications.
